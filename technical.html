Introduction

Python is a high-level, easy-to-learn programming language.

It is widely used in web development, AI, data science, and automation.

print("Hello World")
  • Simple syntax
  • Readable code
  • Large community
Variables and Data Types

Variables store values inside memory.

Python automatically detects data types.

x = 10 name = "Yoha"

Common types include integers, floats, strings, and booleans.

Control Flow

Control flow helps decide how a program runs.

We use conditions and loops.

if x > 5: print("Big number") for i in range(5): print(i)
  • if
  • for
  • while
Functions

Functions group reusable code blocks.

They reduce repetition and improve readability.

def greet(name): return "Hello " + name

Functions can accept parameters and return values.

Object Oriented Programming

OOP organizes programs using classes and objects.

It improves modularity and reuse.

class Student: def __init__(self, name): self.name = name

Concepts include encapsulation, inheritance, and polymorphism.
